Calculator Applications

Resources

There are not as many free resources, as this is a less popular event. I recommend using a RPN calculator, as you can speed through the number crunchers very quickly.

I got a state title (6A Division) using the HP33s, but that calculator is hard to find, so HP35s and HP32SII also works.

I would recommend to just learn how to use RPN. There are many free tutorials on the Internet. Though there is a learning curve, if you can master it you can finish all 58 number crunchers within 10 minutes.

General Math

Resources

General Math is a very speed-based contest, where you get an average of 36 seconds per problem. To master this event, you must know the fundamental concepts (Algebra, Geometry, Number Theory, Combinatorics) and must be able to do it quickly.

I mostly learned this from a helpful website called artofproblemsolving.com, where it has many resoucres aimed specifically for competition math

Helpful Links

Very useful for speed - For the Win! (randomized problems that are very close to general math level, you must answer problems within a certain time limit.)
